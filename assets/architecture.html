PR Pilot Architecture

5-Agent Autonomous Code Review Pipeline — Google Cloud + Gemini API

5-Agent Pipeline

  • • Reviewer: Diff analysis (bugs, security, perf)
  • • Fixer: Minimal patch generation
  • • Tester: Framework detection + test writing
  • • Verifier: Quality gate (no regressions)
  • • Escalator: Auto-approve / request changes / escalate

Google Cloud Stack

  • • Cloud Run: Serverless webhook + orchestration
  • • Gemini API: Powers all 5 agents
  • • Secret Manager: GitHub App keys + API tokens
  • • Cloud Logging: Agent execution audit trail
  • • Cloud Build: CI/CD pipeline

AI-Native Operations

  • • 90%+ reviews handled autonomously
  • • Structured JSON output between agents
  • • Exponential retry on API failure
  • • Full decision audit trail
  • • Human-in-the-loop only for hard cases
